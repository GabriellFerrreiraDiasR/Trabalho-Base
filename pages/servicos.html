HTML

HTML (HyperText Markup Language) é a linguagem responsável pela estrutura das páginas web. Ela organiza textos, imagens, vídeos, links e outros elementos.

Para que serve?

  • Criar páginas web
  • Organizar conteúdos
  • Adicionar formulários
  • Inserir imagens e vídeos
  • Criar links entre páginas

Principais tags

  • header → cabeçalho
  • nav → menu de navegação
  • main → conteúdo principal
  • section → seção
  • article → artigo
  • footer → rodapé

Exemplo:

<h1>Olá Mundo</h1>

<p>
Meu primeiro site
</p>

Curiosidade

HTML foi criado por Tim Berners-Lee em 1991.

CSS

CSS (Cascading Style Sheets) é utilizado para estilizar páginas. Ele controla cores, fontes, tamanhos e posicionamento.

Para que serve?

  • Alterar cores
  • Modificar fontes
  • Criar layouts
  • Fazer animações
  • Deixar páginas responsivas

Principais propriedades

  • color → muda cor do texto
  • background → altera fundo
  • font-size → tamanho do texto
  • margin → espaçamento externo
  • padding → espaçamento interno
  • border-radius → bordas arredondadas

Exemplo:

body{

background:black;

color:white;

font-size:20px;

}

Curiosidade

CSS surgiu em 1996 para separar estilo da estrutura HTML.

JavaScript

JavaScript é uma linguagem de programação usada para adicionar interatividade aos sites.

O que ele pode fazer?

  • Criar menus interativos
  • Validar formulários
  • Modificar elementos da página
  • Criar jogos
  • Consumir APIs
  • Criar animações

Conceitos básicos

  • Variáveis
  • Funções
  • Condições (if)
  • Laços (for e while)
  • Eventos

Exemplo:

let nome = "João";

alert(
"Olá " + nome
);

Curiosidade

JavaScript foi criado em apenas 10 dias por Brendan Eich em 1995.
