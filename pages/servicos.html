HTML

Estrutura de páginas web usando elementos semânticos.

O que é?

HTML (HyperText Markup Language) é a linguagem de marcação que cria a estrutura das páginas web. Criada por Tim Berners-Lee em 1991.

Analogia: É o esqueleto da casa - define onde ficam paredes, portas e cômodos.

Elementos principais

  • Estruturais: header, nav, main, section, article, footer
  • Texto: h1 a h6, p, span, div, strong, em
  • Listas: ul (não ordenada), ol (ordenada)
  • Tabelas: table, tr, td, th
  • Formulários: form, input, select, textarea, button
  • Multimídia: img, audio, video, canvas

Exemplo básico

<!DOCTYPE html>
<html>
<head>
    <title>Meu Site</title>
</head>
<body>
    <h1>Olá Mundo!</h1>
    <p>Meu primeiro parágrafo.</p>
</body>
</html>

Dica: Use tags semânticas para melhorar o SEO e a acessibilidade.

CSS

Estilização, cores e layouts responsivos com Grid.

O que é?

CSS (Cascading Style Sheets) é a linguagem de estilo que define a aparência visual das páginas. Oficial desde 1996.

Analogia: É a decoração da casa - define cores, fontes, tamanhos e posicionamento.

Principais conceitos

  • Sintaxe: seletor { propriedade: valor; }
  • Box Model: margin, border, padding, content
  • Display: block, inline, flex, grid
  • Posicionamento: static, relative, absolute, fixed, sticky
  • Cores: nome, hexadecimal, rgb(), hsl()
  • Unidades: px, %, rem, em, vw, vh

Exemplo básico

/* Seletor por classe */
.minha-classe {
    color: blue;
    font-size: 20px;
    background: #f0f0f0;
    padding: 10px;
    border-radius: 5px;
}

Flexbox (mais usado)

.container {
    display: flex;
    justify-content: center;
    align-items: center;
    gap: 20px;
}

Grid (layout em grade)

.grid {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 15px;
}

Dica: Use sempre box-sizing: border-box no início do seu CSS.

JavaScript

Interatividade e lógica para páginas web.

O que é?

JavaScript é uma linguagem de programação que torna as páginas interativas. Criada por Brendan Eich em 1995.

Analogia: É a eletricidade da casa - permite portas abrirem, luzes acenderem, coisas se moverem.

Conceitos fundamentais

  • Variáveis: let (muda), const (não muda)
  • Tipos: String, Number, Boolean, Array, Object
  • Funções: function nome() { } ou () => { }
  • Condicionais: if, else, switch
  • Loops: for, while
  • Eventos: click, mouseover, keydown, submit

Exemplo básico

// Pegar elementos
const botao = document.getElementById("meuBotao");
const texto = document.querySelector(".mensagem");

// Adicionar evento
botao.addEventListener("click", () => {
    texto.textContent = "Clicou!";
    texto.style.color = "red";
});

Manipulação do DOM (essencial)

// Selecionar
document.getElementById("id")
document.querySelector(".classe")

// Modificar
elemento.textContent = "Novo texto"
elemento.innerHTML = "<strong>HTML</strong>"
elemento.style.color = "blue"
elemento.classList.add("ativo")

Dica: Use const por padrão. Só use let se precisar mudar o valor depois.
